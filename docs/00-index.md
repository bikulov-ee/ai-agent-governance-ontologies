## Текущая рамка проекта

Проект находится на раннем этапе. Верхняя онтология управления AI-агентами пока только зарезервирована, а две специализированные онтологии уже входят в публикуемый корпус.

Следующий содержательный шаг: разработать интеграционный слой "Концептуальная онтология управления AI-агентами", который свяжет допустимость, уместность, автономность, основания и доказуемость агентных действий.
