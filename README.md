Этот репозиторий содержит публикуемый корпус проекта по разработке онтологий управления AI-агентами.

## Состав

- `docs/` — публикационная документация и карта проекта.

Внутренние рабочие материалы, рабочие инструменты агентов и локальные настройки не входят в публикуемый корпус.
